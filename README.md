w2c
(web throw to catch)
=====

__look at notes for crucial information!__

*w2c* is an online service written in JavaScript and HTML, acting as a quick and easy storage space for a multitude of HTML, CSS and JS* files. The aim of w2c is to reduce website storage taken up by these documents by compressing* or translating them. Then, when said documents are requested w2c will decompress* or retranslate them for display on the samm-r.github.io/catch/#/... output site. w2c uses GitHub for it's machinery.

### Usage
to use w2c, you must first register* you and your files on the /throw/ directory of the w2c URL, and when verified you can insert an iframe `<iframe src="https://samm-r.github.io/catch/#/...'` into your index HTML file, but make sure to replace the ellipses in the tag with the name-key you received after registration.
