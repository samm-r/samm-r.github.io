wt2c
====

wt2c (web throw-to-catch) is a service that compress documents thrown (sent to wt2c) into a folder, and then decompress said documents when caught (received) on the wt2c/throw directory. 

wt2c aims to reduce storage taken up by handful of documents and instead displays them on a wt2c iframe.

`throw: WIP`
`catch: -id`
